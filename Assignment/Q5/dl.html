HTML
HTML, or Hypertext Markup Language, is a standard markup language for creating web pages and web applications.
CSS
CSS (Cascading Style Sheets) is a language used for styling web pages, controlling layout, colors, and typography.
Javascript
JavaScript is a versatile programming language primarily used for web development, enabling interactive elements, dynamic content, and client-side scripting within web pages.
MongoDB
MongoDB is a popular NoSQL database, known for its flexibility and scalability, storing data in JSON-like documents. It's widely used for web applications, big data, and real-time analytics.
React
React is a JavaScript library for building user interfaces, developed by Facebook. It uses a component-based architecture and virtual DOM for efficient rendering of UI components.
Express.js
Express.js is a minimal and flexible Node.js web application framework that provides a robust set of features for building web and mobile applications.
Node.js
Node.js is a runtime environment that executes JavaScript code outside of a web browser, enabling server-side scripting for scalable and high-performance web applications.
