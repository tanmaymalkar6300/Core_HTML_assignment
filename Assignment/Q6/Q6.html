1. HTML

    Primary use cases Key features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browser
    • Allows for semantic markup

  2. CSS

    Primary use cases Key features/Benefits
    Styling and layout of web pages
    • Selectors: Define which HTML
      elements to style.
    • Cascading: Style rules can be
      inherited and overridden,
      providing flexibility and modularity.
    • Styles: Control visual aspects like
      color, typography,
      layout, and animations.

  3. Javascript

    Primary use cases Key features/Benefits
    JavaScript is primarily used for
    client-side web development,
    including dynamic content,
    interactivity, form validation, and
    asynchronous communication
    with servers.
    • Dynamic: Supports dynamic content and
      interactions on web pages.
    • Event-driven: Responds to user actions
      like clicks and keyboard inputs.
    • Asynchronous: Executes tasks without
      blocking other operations, enhancing
      performance.
